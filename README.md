# Geolocation Manager

A comprehensive solution for managing and testing geolocation in web browsers.

## Project Structure

```
/
├── extension/           # Chrome extension for geolocation spoofing
│   ├── assets/         # Extension assets and icons
│   ├── manifest.json   # Extension configuration
│   └── README.md      # Extension documentation
├── nextjs-app/         # Next.js application for testing (coming soon)
└── README.md          # This file
```

## Components

### Chrome Extension
A powerful browser extension for managing and overriding your geolocation. Perfect for testing and development purposes.

Key features:
- Override browser geolocation with custom coordinates
- Save and manage favorite locations
- Dark monochrome theme
- Export/Import location profiles

### Next.js Application (Coming Soon)
A web application for testing and verifying geolocation changes.

## Installation

### Chrome Extension
1. Download the latest release from the extension directory
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extracted extension folder

### Next.js App (Coming Soon)
The Next.js application will be available soon.
